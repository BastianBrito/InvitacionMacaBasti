Dress Code

Cóctel

Mujer

Con este dress code podrán lucir un vestido de fiesta corto o midi; ajustado u holgado según su preferencia; estampado para el día o liso para la tarde-noche. Combínenlo con zapatos o sandalias de temporada con tacón, pero apuesten por accesorios delicados, como una cadena finita, pues con los vestidos de fiesta tipo cóctel la idea es no sobrecargarse.

Hombre

Lo correcto es que asistan con un traje con corbata, humita o tirantes, ya sea oscuro o en un color menos tradicional, como gris o azul. Este código otorga esa libertad, así que aprovéchenla, especialmente si están invitados a un matrimonio de día, en temporada de buen tiempo. Completen la tenida con una camisa blanca y zapatos de vestir.
